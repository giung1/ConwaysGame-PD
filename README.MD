# Proyecto de simulacion de Conway's Game of Life
### PEGAR LA CARPETA DONDE ESTA INSTALADO POWER-DEVS, SE CREARA UNA CARPETA ASSETS QUE CONTIENE EL SCRIPT DE INICIALIZACION DEL PDM Y LOS MODULOS EN LAS CARPETAS "VECTOR" Y "DISCRETE"
### Moverse a la carpeta assets y ejecutar el script bash gen_pdm.sh, generara un archivo pdm en examples/gameoflife
### Abrir el archivo con powerdevs y ejecutar la simulacion
### Una vez ejecutada, en scilab, abrir el archivo gameoflife_visualization.sce y ejecutarlo, deberia mostrar la grafica de la simulacion
